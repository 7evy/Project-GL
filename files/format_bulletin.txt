Détail des variables à remplacer dans report.html pour générer un bulletin :

Remplacer $y1 et $y2 par l'année universitaire (2019 et 2020 par exemple).

Remplacer $year par l'année d'études (1, 2 ou 3).

Remplacer $lastname par le nom, $firstname par le prénom, $course par la filière de l'étudiant.



Remplacer $table par le code d'une table HTML :
Pour les UEs :
    <tr>
    	<th> Nom de l'UE </th>
        <th> Note </th>
    </tr>
    
Pour les modules pairs (module 2, 4, 6 de la même UE) :
    <tr>
    	<td align="center" style="padding:4px; border: 1px solid black"> Nom du module </td>
        <td align="center" style="padding:4px; border: 1px solid black"> Note </td>
    </tr>
    
Pour les modules impairs (module 1, 3, 5 de la même UE) :
    <tr>
    	<td align="center" style="background-color:#f0f0f0; padding:4px; border: 1px solid black"> Nom du module </td>
        <td align="center" style="background-color:#f0f0f0; padding:4px; border: 1px solid black"> Note </td>
    </tr>

    <tr>
    	<td style="background-color:white; padding:0px; border: 0px"></td>
    	<td style="background-color:white; padding:0px; border: 0px"></td>
    </tr>
